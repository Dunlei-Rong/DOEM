This is the open source code of our paper: DOEM:ANovelDevelopment-Oriented Evaluation Metric Framework for Service Recommendation
You can find the metrics and loss functions in metrics.py and utils.py. For Programmableweb dataset, the label data in \label and mashup & api embedding, train, val and test data in \data.
We also show an example of method usage in MLP.py.
We recommend you to interagate this files into this code framework: https://github.com/catwinee/LLM-CoSR/tree/main/src.
