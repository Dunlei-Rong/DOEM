This repository provides the open-source code for our paper: DOEM: A Novel Development-Oriented Evaluation Metric Framework for Service Recommendation.

The implementation of the proposed metrics and loss functions can be found in metrics.py and utils.py. For the ProgrammableWeb dataset, the label data are provided in the label/ directory, while the mashup and API embeddings, as well as the training, validation, and test data, are stored in the data/ directory.

We also provide an example of how to use the proposed method in MLP.py.

We recommend integrating these files into the following code framework:
https://github.com/catwinee/LLM-CoSR/tree/main/src

For any questions or further information, please feel free to contact us at rongdunlei@126.com or the email of the first author in our paper.
